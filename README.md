# 유니티를 이용한 학교 투어 프로그램

***
<2019 Interface Programming Exhibition - School Tour 개발>
***

2019년도 인터페이스 프로그래밍 전시회 학교 투어 프로젝트입니다.\
유니티를 기반으로 학교 투어 프로그램을 제작하여 프로그래밍 전시회에 전시하는 것을 목표로 합니다.\
개발기간은 9/2 ~ 12/ 까지 진행됩니다.
